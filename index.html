SwarmMap: Multi-Robot Topographical Surveying Swarm ✔

What?

  • Designed and fabricated a decentralized swarm system consisting of three autonomous rovers for topographical surveying.
  • Implemented dynamic area division (Voronoi partitioning) ensuring fault tolerance; if one rover gets stuck, the remaining robots adapt to complete the mission.
  • Engineered the system to autonomously detect, navigate around, and handle both static and dynamic obstacles in unstructured outdoor environments.

How?

  • Modeled the mechanical chassis and a custom trailing-arm suspension system in SolidWorks, ensuring continuous ground contact and dividing torque evenly across all four wheels.
  • Developed the low-level control loop using an ESP32 microcontroller integrated with Hall effect encoders for precise motor odometry.
  • Deployed ROS2 (Humble) on Ubuntu for high-level autonomous navigation, fine-tuning Nav2 parameters for reliable swarm coordination.

Results

  • Successfully deployed the fully fabricated 3-robot physical swarm in a real-world, off-road testing environment.
  • Achieved resilient, decentralized area coverage, proving the concept's viability for reducing manual surveying time.

Rotary Inverted Pendulum (Furuta) ✔
📄 View Project Report

What?

  • Designed and built a Furuta Pendulum (rotary inverted pendulum) to demonstrate advanced nonlinear control and active balancing mechanisms.
  • Engineered the system to perform an autonomous "swing-up" maneuver from a resting downward position and maintain robust stability in the upright, inverted position against external disturbances.

How?

  • Designed and fabricated the mechanical rotary arm and pendulum linkages, ensuring low-friction joints for accurate dynamic modeling.
  • Integrated high-resolution rotary encoders to precisely measure the angular positions of both the driven horizontal arm and the unactuated vertical pendulum.
  • Developed an energy-based swing-up algorithm, seamlessly transitioning into a custom-tuned Linear Quadratic Regulator (LQR) control scheme to stabilize the pendulum at its peak.

Results

  • Successfully built the physical system and tuned the state-feedback control matrices.
  • Achieved a flawless transition between the nonlinear swing-up phase and the LQR stabilization phase, maintaining a perfect balance with minimal steady-state error.
