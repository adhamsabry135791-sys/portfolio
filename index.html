Driver Drowsiness Safety System ✔
💻 View Source Code

What?

  • Designed and built an RC car equipped with smart safety features, including collision avoidance and a system that automatically slows the car down if the driver exceeds a safe speed limit.
  • Created a camera monitoring system that watches the driver. If it detects they are falling asleep, it safely stops the car, sounds a buzzer, and vibrates a smart bracelet to wake them up.
  • Added an emergency rescue feature that connects to the driver's mobile phone, gets their exact GPS location, and automatically finds the closest hospital.

How?

  • Modeled the vehicle chassis and sensor mounting brackets using SolidWorks.
  • Developed the embedded control loop utilizing an ESP32 microcontroller interfaced with magnetic encoders and ultrasonic sensors for precise speed and distance monitoring.
  • Implemented the high-level computer vision architecture in Python, utilizing a custom-retrained YOLO machine learning model for real-time eye-state detection.

Results

  • Successfully integrated all hardware and software subsystems into a fully functional prototype.
  • The system executed all safety protocols flawlessly in real-world testing, correctly identifying drowsiness and halting the vehicle without delay.
